Немного про элемент dialog:
Можно открыть через свойство open и через js (вызвав методы showModal() и show()).

Диалог открывается в двух режимах, как просто диалог (через open и show()) и как диалог-модалка (через showNodal()) с задним фоном (backdrop).
При открытии как просто диалог, нет задника и закрыть можно только по close() или при submit формы внутри диалога, у которой должен быть 
выставлен атрибут

У него есть атрибут closedby который определяет каким  способом можно закрыть модалку диалога:
по кнопке, по тыку в задник - backdop или при нажатии на esc.

Также есть closeModel
Как можно заблокировать прокрутку страницы при октрытии диалога

body:has(dialog[open]) {
  overflow: hidden;
}
